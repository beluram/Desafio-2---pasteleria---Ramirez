# Amatista Pastelería - Ramirez Belén
El Sitio WEB pertenece a una pastelería de caracter ficticio, la cual en esta edicion ofrece una variedad de productos para que un "cliente" pueda agregar al carrito y finalmente realizar una compra, también posee un formulario para enviar consultas y una página donde el cliente puede iniciar sesión para continuar con su compra. 
El mismo es un proyecto realizado durante el curso de JavaScript del instituto Coderhouse Argentina.
El Sitio WEB contiene las siguientes páginas:

- Inicio 
- Iniciar Sesión
- Carrito

Este proyecto fue realizado con HTML, Javascript, Bootstrap y CSS.
